# Lib.pyl
파이썬 라이브러리에 의존하는 파이썬 코딩 언어

---

# 원리(?)와 기초 문법

import"~~~"으로 라이브러리를 임포트한다.

"라이브러리명"_"함수명"_"전달값" 으로 라이브러리의 함수를 호출한다

end 으로 코드가 끝남을 알린다.

^~~~^ 으로 주석을 작성한다.

---

## 예시 코드
<code>
import"math"

"math"_"plus"_"1+2" ^math 라이브러리야 이건 명령이야. 더해.^

end

</code>

.pyllib 확장자이며, zip 확장자를 번경한 것이다.

```bash
├── pyfuctions
│   ├── 첫번째함수명.py
│   ├── 두번째함수명.py
│   └── 세번째함수명.py
└── info.xml
```
